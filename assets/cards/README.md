# 卡片圖放置說明

把卡片圖放進這個資料夾，**檔名對應卡片編號**，遊戲會自動讀取，程式碼不用改。
圖片不存在時，遊戲會顯示樣式化的占位卡面，所以缺圖也能正常遊玩。

## 檔名對照表

| 檔名 | 卡片 |
|---|---|
| `card_001.png` | 疑惑問號 |
| `card_002.png` | 不急 |
| `card_003.png` | 我還沒上車啊 |
| `card_004.png` | 還有嗎？ |
| `card_005.png` | 摩艾沉默 |

## 圖片規格建議

- 格式：PNG 或 JPG（JPG 的話請把 `data/cards.json` 裡的 `image` 副檔名一併改掉）
- 比例：直式 3:4（例如 768×1024），每張一致最好
- 出牌時會全螢幕特寫演出，解析度建議至少 600px 寬
