# v0.5.5 告白場景精修

## 目標

讓高稀有卡在告白場景有明確特殊待遇。

本版針對：

```text
ev_confession × UR / SSR 卡
```

新增手寫劇情、女主專屬反應、數值分歧與結局傾向。

---

## 新增檔案

```text
data/scene_card_results_v055_confession.json
```

此檔案會由 `prototype/js/data.js` 載入，並合併到 `DATA.scenes[].card_results`。

---

## 資料優先序

目前載入順序：

```text
1. data/scenes.json 原始手寫 card_results
2. data/scene_card_results_phase2.json
3. data/scene_card_results_v055_confession.json
4. data/card_reaction_profiles.json
5. data.js 自動補齊矩陣
```

`scene_card_results_v055_confession.json` 會覆蓋 Phase 2 與自動補齊結果。

## 劇情方向

### 真誠高分線

```text
card_005 摩艾沉默
card_020 教練我想打副本
card_022 自由 Gumayuwei
card_023 蒙娜麗莎 Gumayuwei
```

這些卡會提高好感與真誠，適合導向好結局或曖昧成功結局。

### 搞笑高分線

```text
card_017 空中拉桿
card_018 紅色怒火覺醒
card_019 領域展開
card_021 永不放棄 Gumayuwei
card_025 加餐原始人
```

這些卡會提高搞笑、戰鬥、食慾或自信，適合導向荒謬但不一定失敗的結局。

### 高風險霸總線

```text
card_024 給我擦皮鞋
```

這張卡會大幅提高尷尬、社死與自信，但降低好感與真誠，適合導向壞結局或社死結局。

---

## 後續建議

v0.5.6 可以繼續做：

```text
結局條件卡 × 晚餐店 / 夜市 / 會議室 / 告白
```

重點是讓結局有前因後果，例如：

```text
還有嗎？ → 食慾線
給我擦皮鞋 → 霸總失敗線
摩艾沉默 → 真誠沉默線
我還沒上車啊 → 社死搞笑線
```
