# v0.4 抽卡系統與結局圖鑑規格

## 版本目標

v0.4 的目標是把遊戲從「單輪 Demo」提升為有長期收集動機的原型。

新增四個核心模組：

1. 抽卡系統
2. 卡片收藏
3. 收藏手牌制
4. 結局圖鑑

---

## 1. 抽卡系統

### 1.1 抽卡入口

首頁新增「抽卡」按鈕。

玩家可進入迷因抽卡機，進行：

- 單抽
- 十抽

### 1.2 抽卡機率

| 稀有度 | 機率 |
|---|---:|
| N | 50% |
| R | 30% |
| SR | 13% |
| SSR | 5% |
| UR | 2% |

### 1.3 首次十抽保底

玩家第一次進行十抽時，最後一抽至少為 SR。

目的：

- 讓新玩家快速感受到抽卡爽感
- 避免首次抽卡全部都是低稀有度
- 讓玩家更快擴充可用手牌

### 1.4 抽卡結果

每次抽卡會顯示：

- 卡片稀有度
- 卡片名稱
- 卡片台詞
- 卡片描述
- 是否為 NEW
- 若為重複卡，顯示轉換為多少迷因碎片

---

## 2. 卡片收藏

### 2.1 收藏資料

玩家抽到的新卡會永久加入收藏。

目前使用瀏覽器 `localStorage` 儲存，適合 GitHub Pages Demo。

使用 key：

```text
guma_owned_cards
```

### 2.2 初始收藏

第一次進入遊戲時，玩家至少擁有教學手牌：

```text
card_001 疑惑問號
card_002 不急
card_003 我還沒上車啊
card_004 還有嗎？
card_005 摩艾沉默
```

這樣可以確保第一次遊戲一定可以完整體驗核心劇情。

### 2.3 卡片收藏畫面

首頁新增「卡片收藏」按鈕。

收藏畫面顯示：

- 已收藏數量
- 總卡片數
- 完成度百分比
- 迷因碎片數量
- 所有卡片清單

已解鎖卡片顯示完整資料；未解鎖卡片顯示：

- ？？？？？
- 稀有度
- 類型提示

---

## 3. 重複卡與迷因碎片

### 3.1 重複卡處理

如果抽到已擁有的卡，不會重複加入收藏，而是轉換為迷因碎片。

| 稀有度 | 迷因碎片 |
|---|---:|
| N | 1 |
| R | 3 |
| SR | 10 |
| SSR | 30 |
| UR | 80 |

### 3.2 後續用途

v0.4 先累積迷因碎片，不開放兌換。

v0.5 或 v0.6 可擴充：

- 指定卡兌換
- 卡片升級
- 特殊卡池
- 解鎖特殊結局提示

---

## 4. 收藏手牌制

### 4.1 舊版規則

舊版是從完整牌庫中抽 5 張手牌。

### 4.2 v0.4 新規則

現在改為：

```text
只從玩家已收藏的卡片中抽 5 張手牌
```

目的：

- 讓抽卡與玩法直接連動
- 玩家抽到新卡後，下一輪遊戲就有新選擇
- 收藏越多，路線變化越多

### 4.3 例外規則

如果玩家收藏不足 5 張，系統會自動補齊教學手牌，避免無法開始遊戲。

## 5. 結局圖鑑

### 5.1 解鎖方式

每次玩家走到結局時，系統會自動記錄該結局。

使用 key：

```text
guma_unlocked_endings
```

### 5.2 首頁顯示

首頁會顯示：

- 卡片收集進度
- 結局解鎖進度
- 還有多少個結局未解鎖
- 迷因碎片數

例如：

```text
卡片 12/25
結局 3/13
還有 10 個結局未解鎖
碎片 24
```

### 5.3 圖鑑畫面

首頁新增「結局圖鑑」按鈕。

圖鑑顯示：

- 已解鎖數量
- 結局總數
- 未解鎖數量
- 完成度百分比
- 進度條
- 全結局清單

### 5.4 已解鎖結局

已解鎖結局顯示：

- 結局名稱
- 結局 mood：good / bad / normal
- 結局開頭文字

### 5.5 未解鎖結局

未解鎖結局顯示：

- ？？？？？
- 編號
- 基礎提示

提示會依條件自動產生，例如：

- 和特定卡片使用次數有關
- 有人在公司或公共場合社死了
- 和食物、續攤、吃太多有關
- 真誠值是關鍵
- 提高好感並避免場面失控

## 6. 後續 v0.5 建議

v0.4 已完成基礎收集循環。

v0.5 建議加入：

1. 結局分類：戀愛成功、社死、食物、卡片專屬、隱藏
2. 更精準的未解鎖提示
3. 成就系統
4. 圖鑑篩選與排序
5. 迷因碎片兌換指定卡
6. 卡片池分池：日常池、職場池、運動池、文青池
7. 十抽動畫與稀有卡演出

---

## 7. 設計原則

1. 抽卡不能只是裝飾，必須影響遊戲手牌。
2. 未解鎖數量要明顯顯示，讓玩家有追求目標。
3. 重複卡不能讓玩家覺得浪費，所以轉為迷因碎片。
4. 第一次遊戲不能因為收藏太少而卡住，所以保留教學手牌。
5. 結局圖鑑要讓玩家想重玩，而不是只看結果。
