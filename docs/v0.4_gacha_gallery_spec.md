# v0.4 抽卡、卡片收藏與結局圖鑑規格

## 版本目標

v0.4 的目標是把 Demo 從「單次遊玩」改造成「可重複收集」的版本。

核心新增：

1. 首頁主選單
2. 抽卡系統
3. 卡片收藏
4. 遊戲手牌改為從玩家已收藏卡片中抽取
5. 結局圖鑑
6. 未解鎖結局數量與完成度顯示

---

## 1. 首頁主選單

首頁不再只有「開始遊戲」，改為四個入口：

- 開始上班
- 抽卡
- 卡片收藏
- 結局圖鑑

首頁會顯示玩家目前進度：

```text
卡片 5/25
結局 0/13
還有 13 個結局未解鎖
碎片 0
```

---

## 2. 初始收藏

第一次進入遊戲時，系統自動給玩家教學手牌：

```text
card_001 疑惑問號
card_002 不急
card_003 我還沒上車啊
card_004 還有嗎？
card_005 摩艾沉默
```

原因：

- 保證第一次遊玩不會因為沒抽到核心卡而體驗斷掉
- 保留現有 Demo 的教學路線
- 抽卡系統從第二輪開始影響玩家策略

---

## 3. 抽卡系統

### 3.1 抽卡入口

首頁點「抽卡」進入迷因抽卡機。

目前提供：

- 抽 1 張
- 十連抽

暫時不消耗貨幣，方便 Demo 測試。

---

### 3.2 稀有度機率

```text
N      50%
R      30%
SR     13%
SSR     5%
UR      2%
```

> 目前資料內最高稀有度到 UR，所以暫不實作 LR。

---

### 3.3 新手十連保底

第一次十連抽時，最後一張保底 SR 以上。

判定：

```text
如果 localStorage 沒有 guma_first_ten_done
且本次抽卡數量 >= 10
則第 10 張強制從 SR / SSR / UR 中抽取
```

---

### 3.4 重複卡轉碎片

抽到已擁有卡時，不會新增第二張卡，而是轉換為迷因碎片。

```text
N      +1
R      +3
SR     +10
SSR    +30
UR     +80
```

碎片目前先作為累積資源，預留給後續：

- 指定卡兌換
- 卡片升級
- 卡片皮膚
- 稀有演出解鎖

---

## 4. 卡片收藏

卡片收藏會顯示全部 25 張卡。

已擁有：

- 顯示稀有度
- 顯示卡名
- 顯示台詞
- 顯示描述
- 顯示前 4 個標籤

未擁有：

- 顯示稀有度
- 卡名顯示為「？？？？？」
- 顯示類型提示

收藏頁上方顯示：

```text
已收藏 X/25｜完成度 Y%｜迷因碎片 Z
```

---

## 5. 遊戲手牌規則

v0.4 起，遊戲內手牌不再直接從 25 張完整牌庫抽，而是從玩家已收藏卡片中抽。

### 第一次遊玩

固定手牌：

```text
card_001 ~ card_005
```

### 第二次以後

從玩家已收藏卡片中隨機抽 5 張。

```text
玩家收藏
→ 隨機抽 5 張
→ 進入劇情
```

### 手牌汰換

出牌後淘汰 1 張時，也會從玩家收藏中重抽。

```text
淘汰手牌 A
→ 從收藏中排除目前手牌後抽 1 張
→ 如果收藏數太少，允許抽回剛淘汰的卡
```

## 6. 結局圖鑑

結局圖鑑會顯示目前全部結局。

上方顯示：

```text
已解鎖 X/13
還有 Y 個結局等待探索
完成度 Z%
```

並有進度條。

---

## 7. 結局解鎖

玩家達成結局時，系統將該結局 id 寫入 localStorage：

```text
guma_unlocked_endings
```

重複達成同一結局不會重複計數。

---

## 8. 結局圖鑑顯示規則

### 已解鎖結局

顯示：

- 結局 mood
- 結局標題
- 結局文字第一段

### 未解鎖結局

顯示：

- 未解鎖
- No. 編號
- 標題為「？？？？？」
- 一句提示

提示依條件自動產生：

- 卡片條件 → 和特定卡片使用次數有關
- 社死值 → 有人在公司或公共場合社死了
- 食慾值 → 和食物、續攤、吃太多有關
- 真誠值 → 真誠值是關鍵
- good mood → 提高好感，避免場面失控
- bad mood → 錯誤卡片連續使用
- 其他 → 嘗試不同場景和卡片組合

---

## 9. localStorage key

```text
guma_owned_cards        已擁有卡片 id 陣列
guma_meme_shards        迷因碎片數量
guma_first_ten_done     是否已使用新手十連保底
guma_gacha_count        累計抽卡次數
guma_played             是否已完成第一次遊玩
guma_unlocked_endings   已解鎖結局 id 陣列
```
