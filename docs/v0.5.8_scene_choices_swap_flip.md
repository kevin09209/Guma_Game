# v0.5.8 場景選擇與翻面汰換

## 目標

改善兩個遊戲節奏問題：

1. 每個場景不應只有一張道具卡互動，流程太短。
2. 汰換手牌不應再跳完整抽卡頁，節奏太重。

---

## 場景選擇題

每個場景在 intro 劇情後、道具卡選擇前，會先出現一個簡單選擇題。

流程：

```text
場景 intro
→ 是 / 否 / 選擇題
→ 女主簡短反應
→ 道具卡選擇
```

目前每個場景都先配置 2~3 個簡單選項。

選項會有少量數值影響，例如：

```text
好感
真誠
搞笑
尷尬
社死
食慾
```

選項也會影響女主忍受條。

---

## 緊急任務骰子文字

緊急任務 UI 已簡化成：

```text
1~3
4 Four!
5 水床沒玩到
6 完全法克
```

畫面不再把 4/5/6 的敘述寫得太細。

---

## 汰換牌流程

原流程：

```text
選要汰換的牌
→ 跳抽卡頁
→ 抽 1 張
→ 回劇情
```

新流程：

```text
選要汰換的牌
→ 確認是否汰換
→ 是
→ 原地翻面補新牌
→ 回劇情
```

玩家選牌後不會立刻換掉，必須按「確認汰換」。

---

## 補牌牌庫規則

汰換補牌時，牌庫不能包含目前手牌。

目前規則：

```text
從玩家收藏中抽 1 張不在目前手牌內的卡
```

如果收藏牌庫不足，才會退回全卡池補牌，但仍會優先排除目前手牌。
