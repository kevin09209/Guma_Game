# v0.5.7 緊急救援骰更新

## 目標

修正 v0.5.6 的緊急救援骰規則，讓它更符合搞笑戀愛遊戲的節奏。

本版重點：

```text
女主忍受度歸 0
→ Gumayuwei 大喊「等等！！！」
→ 進入緊急救援環節
→ 選 SSR / UR 救援卡
→ 擲 1~6 骰
```

---

## 忍受度歸 0 的演出

當女主忍受條歸 0 時，不會直接跳緊急任務 UI。

會先播放一段男主台詞：

```text
Gumayuwei：等等！！！
```

然後才進入 `EMERGENCY MISSION`。

---

## 新骰子規則

### 1~3：成功救援倍率

如果擲出 1~3，會把「該救援卡在目前場景對應的效果」乘上該點數。

```text
1 點：該卡本場景效果 ×1
2 點：該卡本場景效果 ×2
3 點：該卡本場景效果 ×3
```

注意：這裡不是用卡片基礎效果，而是用：

```text
目前場景 card_results 裡該卡的 effects
+ 該場景的 tag bonus
```

也就是同一張救援卡在不同場景會有不同救援效果。

---

### 4~6：直接壞結局

如果擲出 4~6，救援失敗並直接進入緊急任務壞結局。

```text
4 Four!：準備回家自己爽了
5 水：水床沒玩到，那就都不要玩了
6 完全法克：大法師完全搞砸了
```

---

## 壞結局

本版更新後的緊急壞結局：

```text
沒有救援卡的夜晚
Four! 回家自己爽了
水床沒玩到
完全法克
```

這些目前仍是即時壞結局，尚未加入結局圖鑑。

---

## 成功救援後

擲出 1~3 時，會：

1. 讀取救援卡在當前場景的效果
2. 乘上骰子點數
3. 套用到本局數值
4. 根據救援力回復女主忍受條
5. 繼續遊戲

忍受條會回復到：

```text
12% ~ 70%
```

---

## 平衡建議

目前骰子成功率：

```text
1~3 成功：50%
4~6 失敗：50%
```

這個成功率很刺激，但偏硬派。

如果之後覺得壞結局太常發生，可以調整成：

```text
1~4 成功
5~6 失敗
```

或加入道具：

```text
消耗 1 張抽卡券重骰
消耗 10 迷因碎片重骰
UR 救援卡可以重骰一次
```
